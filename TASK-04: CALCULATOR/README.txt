# SimpleCalculator
This is a Simple Calculator Made using HTML, CSS, JavaScript.

-> Firstly, open a new Directory on your Drive.
-> Open the main.html in this page and copy the content.
-> Go to the Notepad and paste the content in .html file there.
-> Now save the file as "calculator.html" and the type="all types".
-> Do same to the .css and .js files.
-> Now open the main.html in your Browser.
-> you can see the output and test all the Cases.
