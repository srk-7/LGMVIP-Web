# Student-Registration-form

This is a webpage where the student fills the form and the form data will be printed in the same webpage as a idcard. HTML, CSS, JavaScript are used to build this Webpage.

-> Download the .html,.js,.css files from this repository.
-> Make sure that you will place them in same Directory in your system.
-> Now open the .html file.
-> You can see the interface which is divided into two halves one for form and another for the data in the form.
-> Now Fill all the details in the form and press on Enroll.
-> Make sure that you will place the url of the image.
-> You can see that the Form details are fetched and Display over there.
-> In this way you can add n number of data and display them.
